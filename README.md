# Project

This project was generated with [Pruvious](https://pruvious.com) CLI version 3.11.7.

Documentation: [https://pruvious.com/docs](https://pruvious.com/docs)

## Setup

Make sure to install the dependencies:

```bash
# pnpm
pnpm install

# npm
npm install
```

## Development server

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm dev

# npm
npm run dev
```

## Production

Build the website for production:

```bash
# pnpm
pnpm build

# npm
npm run build
```

Check out the [deployment documentation](https://pruvious.com/docs/deployment) for more information.
